    API documentation

    Module Description
    raijin.appconfig This module combines the functionality of VariantConfig and configpath into one class.
    raijin.arrayutils Various functions for working with arrays.
    raijin.commandlineargs This module contains a simple class that processes command line arguments and stores them in a hashtable. The hash table is static so this class object can be created multiple times without having to reinitialize it when the arguments from main.
    raijin.configpath Provides an easy way to work the users configuration directory.
    raijin.debugutils Functions for outputing a debug string.
    raijin.fileutils Various functions for dealing with files.
    raijin.pathutils Various functions for dealing with path based manipulation and retrieval.
    raijin.stringutils This module contains a few additons to handling strings.
    raijin.typeutils Various functions to determine types at runtime.
    raijin.variantconfig This module manages a config file format in the form of key=value. Much like an ini file but simpler.
