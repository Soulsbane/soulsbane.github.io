    API documentation

    Module Description
    raijin.utils.array Various functions for working with arrays.
    raijin.utils.debugtools Functions for outputing a debug string.
    raijin.utils.file Various functions for dealing with files.
    raijin.utils.path Various functions for dealing with path based manipulation and retrieval.
    raijin.utils.string This module contains a few additons to handling strings.
    raijin.utils.type Various functions to determine types at runtime.
    raijin.appconfig This module combines the functionality of KeyValueConfig and ConfigPath into one class.
    raijin.cmdline A simple module that handles user input on the command line.
    raijin.commandlineargs This module contains a simple class that processes command line arguments and stores them in a hashtable. The hash table is static so this class object can be created multiple times without having to reinitialize it when the arguments from main.
    raijin.configpath Provides an easy way to work the users configuration directory.
    raijin.keyvalueconfig This module manages a config file format in the form of key=value. Much like an ini file but simpler.
    raijin.timers Add support for creating a repeating timer.
