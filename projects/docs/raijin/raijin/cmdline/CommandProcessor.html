    Class CommandProcessor

    Manages a loop which processes commands via command line input.

    Inherits from

    • Object (base class)

    Methods

    Name Description
    addCommand Adds a command that should be processed.
    isValidCommand Determines if a command is valid(added via addCommand).
    onCommand Recieves commands sent from processCommands
    onEnterProcessCommands Called by processCommands after commands are handled.
    onExitProcessCommands Called by processCommands before commands are handled.
    onListCommands Called by processCommands when the "list" command is sent.
    processCommands Processes commands sent via the command line.
    quit Called by processCommands when the "exit" command is sent.

    Examples

    class MyCommand : CommandProcessor
{
	// All on<name> commands can be overriden.
	override void onCommand(const string command, const string[] args)
	{
		// Will only print "test" since it is the only command registered.
		writeln("Command: ", command, " Args: ", args);
	}
}

auto cmd = new MyCommand;

cmd.addCommand("test", "this is a test");
cmd.processCommands();
