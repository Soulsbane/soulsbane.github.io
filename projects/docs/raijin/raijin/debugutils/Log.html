    Struct Log

    Wrapper around info, warn and error functions that acts as a namespace.

    Methods

    Name Description
    error Outputs a message to stdout marked as type error with function name, line number, time and file name attached.
    info Outputs a message to stdout marked as type info with function name, line number, time and file name attached.
    warn Outputs a message to stdout marked as type warn with function name, line number, time and file name attached.

    Example

    Log.info("This is a info test");
Log.warn("This is a warn test");
Log.error("This is a error test");
