    Function CommandLineArgs.safeGet

    Retrieves the raw value passed via the command line in a safe way.

    Prototype

    final T safeGet(T)(
      const size_t index,
      string defaultValue = string.init
    ) @safe;

    Parameters

    NameDescription

    defaultValue Default value to use if the index is out of range.
