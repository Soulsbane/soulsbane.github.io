    Class CommandLineArgs

    Handles the processing of command line arguments.

    Inherits from

    • Object (base class)

    Methods

    Name Description
    addFlag Registers a flag argument eg. --flag
    contains Determines if the key(command line argument) exists.
    getProgramName Returns the name of the program.
    onInvalidArg Should not be called directly. Called when an invalid argument is passed on the command line.
    onNoArgs Should not be called directly. Called when no arguments were passed on the command line.
    onNonArg Should not be called directly. Called when when an argument does not contain a leading '-' and ignoreNonArgs is enabled.
    onPrintHelp Should not be called directly. Default print method for printing registered command line options.
    onPrintVersion Should not be called directly. Prints the program version string when -version argument is passed.
    onValidArg Should not be called directly. Called each time a valid argument is passed.
    onValidArgs Should not be called directly. Called after all arguments have been processed and no invalid arguments were found.
    opIndex Retrieves the value of key where key is the name of the command line argument. T is the the type that returned value should be converted to.
    opIndexAssign Assigns a value to a commandline argument stored internally.
    opIndexAssign Assigns a value to a commandline argument stored internally but uses ArgValues as the value.
    process Handles the registration of command line arguments passed to the program.
    safeGet Retrieves the raw value passed via the command line in a safe way.
    setProgramVersion Sets the programs version string to be used with -version argument.

    Example

    auto arguments = ["testapp", "-flag", "-value=this is a test", "-aFloat=4.44"];
auto args = new CommandLineArgs;

args.addFlag("flag", "A test flag");
args.addCommand("aFloat", 3.14, "A float value");
args.addCommand("integer", 100, "Sets value");
args.addCommand("value", "hello world", "Just a silly value.");
args.process(arguments);

assert(args.asBoolean("flag") == true);
assert(args["flag"] == true);

import std.math;

float aFloat = args.asDecimal("aFloat");

assert(feqrel(args.asDecimal("aFloat"), 4.44));
assert(feqrel(aFloat, 4.44));

assert(feqrel(args.asInteger("aFloat"), 4.44));

assert(args.contains("integer") == true);
assert(args.contains("valuez") == false);
assert(args["integer"] == 100);
