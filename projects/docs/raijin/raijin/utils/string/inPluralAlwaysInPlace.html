    Function inPluralAlwaysInPlace

    Pluralizes a string in place.

    Prototype

    void inPluralAlwaysInPlace(
      ref string word,
      const(string) pluralizeToWord = null
    ) pure @safe;

    Parameters

    NameDescription
    word The word to pluralize.
    pluralizeToWord The word to use when a value needs to be pluralized

    Example

    string cat = "cat";
cat.inPluralAlwaysInPlace("catz");
assert(cat == "catz");
