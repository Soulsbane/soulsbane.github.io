    Class RepeatingTimer

    A class for creating a RepeatingTimer.

    Inherits from

    • Object (base class)

    Properties

    Name Type Description
    name [get] string Returns the name of the timer(this.toString by default).
    name [set] const(string) Sets the threads name.

    Methods

    Name Description
    onTimer Called each every Duration specified.
    onTimerStart Called when the timer starts.
    onTimerStop Called when the timer stops.
    setCallBack Sets a callback to a function instead of having to inherit from class.
    start Starts the timer
    stop Stops the timer.

    Examples

    import std.stdio;
import raijin;

class MyRepeatingTimer : RepeatingTimer
{
	this()
	{
		name = "This is the name MyRepeatingTimer";
	}
	override void onTimer()
	{
		writeln(name);
	}
}

class InitialDelayRepeatingTimer : RepeatingTimer
{
}

void main()
{
	auto myDelay = dur!("seconds")(1);

	auto myTimer = new MyRepeatingTimer;

	myTimer.start(myDelay);

	auto initialDelayTimer = new InitialDelayRepeatingTimer ;
	auto initialDelay = dur!("seconds")(3);

	initialDelayTimer.start(myDelay, initialDelay);

	auto delay = dur!("seconds")(1);
	auto countdownDelay = dur!("seconds")(3);

	writeln("HELLO");
}
