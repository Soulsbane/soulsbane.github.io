    Struct SimpleQueue

    A simple implementation of a queue.

    Template arguments

    struct SimpleQueue(T);

    Methods

    Name Description
    all Returns the entire queue array.
    clear Clears the queue.
    empty Determines if the queue is empty.
    front Retrieves the value at the front of the queue(the first value pushed).
    length Retries the number of items in the queue;
    pop Pops the value at the queues back off the queue.
    push Pushes a value onto the queue.

    Example

    SimpleQueue!int queue;

queue.push(1);
queue.push(2);
queue.push(3);

foreach(value; queue.all) {}

assert(queue.length == 3);
assert(queue.pop() == 1);
assert(!queue.empty);
assert(queue.front == 2);

queue.clear();
assert(queue.length == 0);
