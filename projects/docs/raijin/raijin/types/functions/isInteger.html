    Function isInteger

    Determines if a string is an integer value.

    Prototype

    bool isInteger(
      const(string) value
    ) pure @safe;

    Parameters

    NameDescription
    value string to use.

    Returns

    true if the value is a integer false otherwise.

    Example

    assert("13".isInteger == true);
assert("13.333333".isInteger == false);
assert("zzzz".isInteger == false);
