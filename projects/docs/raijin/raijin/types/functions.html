    Module raijin.types.functions

    Various functions to determine types at runtime.

    Functions

    Name Description
    isBoolean Determines if a value is of type boolean using 0, 1, true and false as qualifiers.
    isDecimal Determines if a string is a decimal value
    isFalse Determines if value is a false value
    isInteger Determines if a string is an integer value.
    isTrue Determines if value is a true value
    list Equivalent of PHP's list language construct: Works with arrays and tuples. Specify null as an argument to ignore that index (equivalent of list(x, , y) in PHP).
